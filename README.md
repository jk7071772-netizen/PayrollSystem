# Payroll Management System

ASP.NET Core MVC mein bana hua complete Payroll System.

## Features:
✅ Employee Management (CRUD)
✅ Attendance with Check-in/Check-out
✅ Leave Management with Approval
✅ Loan Management with EMI
✅ Payroll Processing
✅ PDF Pay Slips
✅ Reports with Charts
✅ Admin, HR, Employee Dashboards

## Tech Stack:
- ASP.NET Core MVC 8.0
- C# / .NET Core
- Entity Framework Core
- SQL Server
- Bootstrap 5
- JavaScript / jQuery / AJAX
- Chart.js
